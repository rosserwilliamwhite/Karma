# Shithead

Codification of shithead to optimise strategy

# Aims
Learn optimal strategy for shithead by seeing what an AI learns over tens of thousands of games
Is it optimal to pick up?
Is it optimal to play a higher card to force your opponent to pick up?

# Shithead Rules

1. Players must only place cards of equal or higher rank. Any number of that and only that card can be played. If you only have cards of lower rank, pick up the discard.
2. After each sucessful turn refill cards to three
3. If the last four cards in the discard pile are the same, clear the discard and go again
4. If the draw is empty play from table cards. Only play one at a time.
